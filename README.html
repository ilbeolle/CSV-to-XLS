Excel/CSV to XLS Converter (v2.1.0)

프로그램 요약

이 프로그램은 최신 Excel 파일(.xlsx, .xls) 또는 데이터 파일(.csv)을 구형 Excel 97-2003 형식(.xls)으로 변환하는 자동화 도구입니다. 네이버 스마트 스토어, ESM Plus, 현대 이지웰 등 E-커머스 플랫폼 호환성을 위해 제작되었으며, 비밀번호(1234) 자동 해제, 중복 파일명 자동 처리, 시스템 로컬 경로 정밀 탐지 기능을 갖추고 있습니다.

1. 지원 플랫폼 및 기능

  • 네이버 스마트 스토어: 주문/배송 관리 파일 (비밀번호 1234 자동 해제)
  • ESM Plus (G마켓, 옥션): 발송 관리 데이터 변환
  • 현대 이지웰 복지몰: CSV 파일 변환
  • 기타: UTF-8, EUC-KR 인코딩 CSV 및 XLSX 데이터

v2.1.0 업데이트 사항

  • 스마트 경로 탐지: Windows 레지스트리를 조회하여 실제 바탕화면 경로를 찾아냅니다. (OneDrive 오류 방지)
  • 충돌 방지 시스템: 파일명 중복 시 파일명_1.xls 형태로 자동 저장하여 덮어쓰기를 방지합니다.
  • 안정성 강화: 프로세스 종료 로직을 개선하여 Excel이 백그라운드에 남지 않도록 설계되었습니다.

2. 사용 방법

방법 A: 자동 모드 (가장 쉬움)

  1. 변환할 파일(xlsx, csv 등)을 마우스로 선택합니다.
  2. 선택한 파일을 ExcelConverter.exe 아이콘 위로 드래그 앤 드롭합니다.
  3. 검은색 창이 잠시 나타났다 사라지며, 바탕화면에 변환된 파일이 생성됩니다.

방법 B: 대화형 모드 (고급)

  1. ExcelConverter.exe를 실행하여 검은색 창을 엽니다.
  2. 변환하고 싶은 파일을 창 안으로 끌어다 놓고 Enter를 누릅니다.
  3. 작업이 완료되면 바탕화면 폴더가 자동으로 열립니다.
  4. 종료하려면 exit를 입력하세요.

3. 문제 해결 (FAQ)

주의: Microsoft Excel 필수
이 프로그램은 Microsoft Excel 정품이 PC에 설치되어 있어야 정상 작동합니다. (한셀, 리브레오피스 호환 불가)
  • Windows Defender 경고창이 뜰 때: "추가 정보" -> "실행"을 클릭하거나, 백신 예외 항목에 폴더를 추가하십시오. (개인 개발자 서명 미포함으로 인한 오진입니다.)
  • OneDrive 오류: 파일 경로를 찾지 못할 경우, 파일을 바탕화면이나 내 문서가 아닌 C:\Downloads 등의 로컬 폴더로 옮겨서 시도하십시오.

4. [전문가용] 실행 파일 빌드 가이드

이 섹션은 소스 코드(converter.py)를 기반으로 EXE 파일을 직접 생성하려는 개발자를 위한 안내입니다.

개발 환경 준비

  • OS: Windows 10/11 (x64)
  • Python: 3.8 이상 권장
  • 필수 라이브러리 설치:
pip install pyinstaller colorama pywin32

EXE 생성 명령어

프로젝트 폴더에서 터미널을 열고 아래 명령어를 입력하십시오. (아이콘 파일 icon.ico가 있을 경우)

pyinstaller --noconfirm --onefile --console --clean --name "ExcelConverter" --icon "icon.ico" converter.py

옵션 설명:

  • --onefile: 하나의 EXE 파일로 압축
  • --console: 오류 확인을 위한 콘솔 창 유지
  • --clean: 캐시 삭제로 빌드 오류 방지
